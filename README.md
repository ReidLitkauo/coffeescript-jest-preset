# Coffeescript jest preset

Jest preset for Coffeescript.

## How to use

```bash
npm install coffeescript-jest-preset --save-dev
```

In your jest settings add `coffeescript-jest-preset` preset.

```json
{
  "preset": "coffeescript-jest-preset"
}
```
